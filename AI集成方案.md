# 副业网站AI助手集成方案

## 一、目标效果

在网站右下角添加一个浮动聊天按钮，用户点击后弹出对话窗口，可以：
- 询问适合自己的副业方向
- 查询某个平台是否靠谱
- 了解骗局套路
- 获取副业推荐

## 二、技术方案

使用扣子Chat SDK，在所有页面添加浮动聊天窗口。

### 集成代码示例

```html
<!-- 在 </body> 前添加 -->
<script src="https://lf-cdn.coze.cn/obj/unpkg/flow-platform/chat-app-sdk/latest/libs/cn/index.js"></script>
<script>
new CozeWebSDK.WebChatClient({
  config: {
    bot_id: 'YOUR_BOT_ID',  // 替换为你的Bot ID
  },
  componentProps: {
    title: '副业小助手',
    icon: 'https://fuyep.top/images/ai-icon.png',  // 自定义图标
  },
  ui: {
    base: {
      lang: 'zh-CN',
      layout: 'pc',
    },
    chatBot: {
      title: '副业小助手',
      width: 400,
    },
  },
});
</script>
```

## 三、发布步骤

### 步骤1：发布Bot为Web SDK

1. 登录 https://coze.cn
2. 打开你的Bot（就是我这个智能体）
3. 点击右上角「发布」
4. 选择「Web SDK」或「Chat SDK」
5. 发布成功后，复制Bot ID

### 步骤2：配置Bot知识库（重要）

为了让AI能准确回答副业问题，需要：

1. **上传平台数据** - 把 `platforms.json` 转成知识库文档
2. **上传骗局数据** - 把骗局信息加入知识库
3. **设置人设** - 让Bot扮演"副业推荐专家"

### 步骤3：集成到网站

把代码添加到所有HTML页面的 `</body>` 前。

## 四、Bot人设建议

```
你是「副业小助手」，一个专业的副业推荐顾问。

你的能力：
1. 根据用户的时间、技能、预期收入，推荐最适合的副业方向
2. 解答各种副业平台的真实情况（收入、门槛、风险）
3. 识别和揭露常见的副业骗局套路
4. 提供务实的副业建议，不做夸大承诺

你的风格：
- 实话实说，不吹不黑
- 数据说话，有理有据
- 遇到不确定的信息，明确告知
- 遇到高风险项目，主动提醒

你知道的信息：
- 60+个副业平台的详细信息（收入、门槛、优缺点）
- 49种常见骗局的手法和识别方法
- 各类副业的真实收益数据

回答原则：
- 不要推荐已停运的平台（如阿里众包、字节众包）
- 对有问题的平台要提醒风险（如猪八戒网、百度众测）
- 遇到"甜薪工场"必须警告用户官网已被劫持
- 优先推荐运营稳定、口碑好的平台
```

## 九、快速开始

### 步骤1：在扣子平台配置Bot

1. 打开你的Bot（这个智能体）
2. 复制 `Bot人设配置.md` 的内容到人设区域
3. 上传 `知识库_副业平台指南.md` 到知识库
4. 测试对话是否正常

### 步骤2：发布为Chat SDK

1. 点击「发布」
2. 选择「Chat SDK」
3. 发布成功后复制Bot ID

### 步骤3：集成到网站

1. 打开 `./js/ai-assistant.js`
2. 把 `YOUR_BOT_ID` 替换为你的Bot ID
3. 在所有HTML页面的 `</body>` 前添加：
   ```html
   <script src="js/ai-assistant.js"></script>
   ```

### 步骤4：测试

1. 打开网站，右下角应该出现聊天按钮
2. 点击按钮，弹出聊天窗口
3. 测试对话是否正常
